1. Basic Python Concepts

Python is a versatile and beginner-friendly programming language. Here's how you can use Python to perform basic operations, create variables, and work with loops.

Example: Variables and Operations

# Example: Variables and Operations
x = 10
y = 5
sum_xy = x + y
print(f"Sum of {x} and {y}: {sum_xy}")

Conditionals Example

# Example: Conditionals
if x > y:
    print(f"{x} is greater than {y}")
else:
    print(f"{y} is greater than {x}")

Loops Example

# Example: For Loop
for i in range(3):
    print(f"Loop iteration {i}")

# Example: While Loop
count = 0
while count < 3:
    print(f"Count is {count}")
    count += 1

2. Functions and Object-Oriented Programming (OOP)

Functions help in organizing the code into reusable blocks. In Object-Oriented Programming (OOP), classes and objects help in organizing the data and behavior.

Example: Defining a Function

# Example: Function
def greet(name):
    return f"Hello, {name}!"
        
print(greet("Alice"))

Example: Object-Oriented Programming (OOP)

# Example: OOP with Dog Class
class Dog:
    def __init__(self, name, breed):
        self.name = name
        self.breed = breed

    def speak(self):
        print(f"{self.name} says Woof!")

dog1 = Dog("Buddy", "Golden Retriever")
dog1.speak()

3. Advanced Python: File Handling and Error Handling

Python allows you to interact with files, handle errors, and perform operations that prevent the program from crashing.

Example: File Handling

# Example: File Handling (Reading and Writing)
with open("example.txt", "w") as file:
    file.write("Python is amazing!")
        
with open("example.txt", "r") as file:
    content = file.read()
    print(f"File Content: {content}")

Example: Error Handling

# Example: Error Handling
try:
    num = int(input("Enter a number: "))
    result = 10 / num
    print(f"Result: {result}")
except ZeroDivisionError:
    print("Cannot divide by zero!")
except ValueError:
    print("Invalid input. Please enter a number.")

4. Real-Life Applications: AI, Speech Recognition, and Automation

Python is widely used in building applications for AI, speech recognition, web automation, and more. Below are examples demonstrating these applications.

Example: Speech-to-Text (Using Speech Recognition)

# Example: Speech-to-Text (requires speech_recognition library)
import speech_recognition as sr

recognizer = sr.Recognizer()
with sr.Microphone() as source:
    print("Listening for speech...")
    audio = recognizer.listen(source)
    try:
        text = recognizer.recognize_google(audio)
        print(f"You said: {text}")
    except sr.UnknownValueError:
        print("Sorry, I did not understand that.")
    except sr.RequestError:
        print("Error with the speech recognition service.")

Example: Text-to-Speech (Using pyttsx3)

# Example: Text-to-Speech (requires pyttsx3 library)
import pyttsx3

engine = pyttsx3.init()
engine.say("Hello, I am your assistant.")
engine.runAndWait()

Example: Building an AI Chatbot

# Example: AI Chatbot (requires OpenAI API key)
import openai

openai.api_key = 'YOUR_API_KEY'

response = openai.Completion.create(
    engine="text-davinci-003",
    prompt="Hello, how can I help you?",
    max_tokens=100
)
print(response.choices[0].text.strip())

Example: Web Automation (Opening a Website)

# Example: Web Automation (using webbrowser module)
import webbrowser

5. Data Analysis and Visualization

Python is one of the leading languages for data analysis and visualization using libraries such as pandas and matplotlib.

Example: Data Analysis with Pandas

# Example: Data Analysis using pandas
import pandas as pd

data = {"Name": ["Alice", "Bob", "Charlie"], "Age": [25, 30, 35]}

Example: Data Visualization with Matplotlib

Visualizing data using Python is easy with libraries like matplotlib.

# Example: Data Visualization using matplotlib
import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 100)
y = np.sin(x)

plt.plot(x, y)
plt.title("Sine Wave")
plt.xlabel("x")
plt.ylabel("sin(x)")
plt.show()

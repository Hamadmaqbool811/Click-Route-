1. Basic Python Concepts

Python is a versatile and beginner-friendly programming language. Here's how you can use Python to perform basic operations, create variables, and work with loops.

Example: Variables and Operations

# Example: Variables and Operations
x = 10
y = 5
sum_xy = x + y
print(f"Sum of {x} and {y}: {sum_xy}")

Conditionals Example

# Example: Conditionals
if x > y:
    print(f"{x} is greater than {y}")
else:
    print(f"{y} is greater than {x}")

Loops Example

# Example: For Loop
for i in range(3):
    print(f"Loop iteration {i}")

# Example: While Loop
count = 0
while count < 3:
    print(f"Count is {count}")
    count += 1

Functions help in organizing the code into reusable blocks. In Object-Oriented Programming (OOP), classes and objects help in organizing the data and behavior.

Example: Defining a Function

# Example: Function
def greet(name):
    return f"Hello, {name}!"

Example: Object-Oriented Programming (OOP)

# Example: OOP with Dog Class
class Dog:
    def __init__(self, name, breed):
        self.name = name
        self.breed = breed

    def speak(self):
        print(f"{self.name} says Woof!")

dog1 = Dog("Buddy", "Golden Retriever")
dog1.speak()

3. Robotics and Automation with Python

Python can be used to program robots or automated systems. Below is a simple example where we simulate a robot performing tasks.

Example: Robot Arm Movement Simulation

Hereâ€™s an example using Python to simulate controlling a robot's arm. We will use the `pyserial` library to communicate with the robot hardware, and the example shows simple motor control logic.

# Example: Simulating a Robot Arm Movement

import serial
import time

# Simulate connection to robot arm via serial port (change 'COM3' to your port)
ser = serial.Serial('COM3', 9600, timeout=1)

# Move the robot arm to specific positions
def move_arm(position):
    ser.write(position.encode())
    time.sleep(1)

# Simulate different movements
move_arm("MOVE UP")
move_arm("MOVE DOWN")
move_arm("STOP")

Note: For actual robot control, you would need to connect to real hardware and send commands via serial or another communication protocol. You can use ROS (Robot Operating System) for more advanced projects.

Example: Automated Task (Turn On/Off a Light)

This example simulates controlling an automated system (like a light) with Python.

# Example: Automated Task (Using GPIO pins on Raspberry Pi)
import RPi.GPIO as GPIO

# Set up GPIO pin (e.g., GPIO pin 17)
GPIO.setmode(GPIO.BCM)
GPIO.setup(17, GPIO.OUT)

# Turn on the light (set pin HIGH)
GPIO.output(17, GPIO.HIGH)

# Wait for 5 seconds
time.sleep(5)

# Turn off the light (set pin LOW)
GPIO.output(17, GPIO.LOW)

# Clean up GPIO
GPIO.cleanup()

4. Real-Life Applications: AI, Speech Recognition, and Automation

Python is widely used in building applications for AI, speech recognition, web automation, and more. Below are examples demonstrating these applications.

Example: Speech-to-Text (Using Speech Recognition)

# Example: Speech-to-Text (requires speech_recognition library)
import speech_recognition as sr

recognizer = sr.Recognizer()
with sr.Microphone() as source:
    print("Listening for speech...")
    audio = recognizer.listen(source)
    try:
        text = recognizer.recognize_google(audio)
        print(f"You said: {text}")
    except sr.UnknownValueError:
        print("Sorry, I did not understand that.")
    except sr.RequestError:
        print("Error with the speech recognition service.")

Example: Text-to-Speech (Using pyttsx3)

# Example: Text-to-Speech (requires pyttsx3 library)
import pyttsx3

engine = pyttsx3.init()
engine.say("Hello, I am your assistant.")
engine.runAndWait()
